00316 /*
00317   Rules are defined per port. Each redirector device
00318   has 4 ports (numbeder 0..3):
00319 
00320          0   +--------------+   2   +--------------+
00321   LAN  <===> |              | <===> |   1/10G      |
00322              |  Redirector  |       |   Ethernet   |
00323   LAN  <===> |    Switch    | <===> |   Adapter    |
00324          1   +--------------+   3   +--------------+
00325 
00326   Drop Rule
00327   Discard incoming packets matching the filter 
00328   on 'rule_port'
00329 
00330   Redirect Rule
00331   Divert incoming packets matching the filter 
00332   on 'rule_port' to 'rule_target_port'.
00333 
00334   Mirror Rule
00335   Copy incoming packets matching the filter 
00336   on 'rule_port' to 'rule_target_port'. The original
00337   packet will continue its journey (i.e. packet are
00338   actually duplicated)
00339 */
