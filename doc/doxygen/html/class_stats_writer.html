Detailed Description

Todo:
Track this down and fix.

This class maintains counters, averages, etc. for tracking the statistics of a monitored object. The monitored object periodically calls the update() method which then updates the statistics counters etc. and then updates instantaneous, exponentially averaged, and long term statistics. Periodically, the data are written out to a CSV file with a name that is unique to the monitored object.
