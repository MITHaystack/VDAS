00160  * UDP port range used for ddp-in-udp encapsulation is 16512-16639
00161  * for client sockets (128-255) and 200-327 for server sockets
00162  * (0-127).  We also try to recognize the pre-April 88 server
00163  * socket range of 768-895.
00164  */
00165 #define atalk_port(p) \
